# AuthentikATOR

Programme d'authentification à double facteurs TOTP pour la platforme Alizon.

## Documentation 

## Lancement

Pour commencer, l'utilisateur sera rediriger vers A2F-pres.php, qui lui présentera l'utilité de la double authentification.
Elle pourra ensuite continuer en se redirigeant vers A2F-code.php, qui lui présentera une clé suivi d'un QR code.
